AutoUpdate Guide

How to use AutoUpdate

Follow these steps to use the AutoUpdate feature:

  1. Download the SaveSync mod here.
  2. Install the mod following the instructions here.
  3. In CarDex, navigate to AutoUpdate > Pick Location
  4. Navigate to Documents/CarDealerSimulator directory
  5. Select the SaveSync folder and confirm.
  6. The CarDex will now automatically update, whenever you save your game.
